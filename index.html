Informations recensées

Cette page contient des informations recensées dans le code source de toutes les pages du site de la Branche francophone de la Fondation SCP. Elle vise à permettre une vision d'ensemble et une réflexion sur la base de code, les styles et inclusions et autre usages de ces pages difficiles à évaluer directement via Wikidot.

De plus, si vous téléchargez l'ensemble de données JSON (voir ci-dessous), vous pourrez utiliser le script grep.py pour chercher à travers le code source de toutes les pages avec quelques commandes simples.

Voir le dépôt GitHub pour plus d'informations.
